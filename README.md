# Livraria Backend (esqueleto)

Estrutura inicial do backend em TypeScript.

Como rodar (dev):

1. Instale dependências: `npm install`
2. Rode em dev (ex: com `ts-node-dev`): `npm run dev`

Estrutura criada:
- `src/controllers`
- `src/models`
- `src/services`
- `src/routes`
- `src/dtos`
- `src/config`
- `src/admin`

Próximos passos: escolher ORM (Prisma / TypeORM / Mongoose) e instalar dependências.
